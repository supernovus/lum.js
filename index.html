Lum[V5] Test

Testing must be done via the JS console. The global-object compatibility library is available as the Lum global variable. It includes the following modular libraries, wrapped in an object emulating the v4 implementation:

core, wrapper, simple-loader, jquery-plugins, dom, compat/v4.

Use Lum.load to load more libraries. In addition to the main libraries in scripts/nano, there are also several scripts available in scripts/ext:

clipboard, easel, feature, jquery, jquery-ui, luxon, moment, require, sprintf, stacktrace.
