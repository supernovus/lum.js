Lum[V5] Test

Testing must be done via the JS console. The global-object compatibility library is available as the Lum global variable, and a custom library testing helper is available as the lt global variable.

Use l.lum(...libs) to load more Lum.js libraries.

In addition to the main libraries in scripts/nano, there are also several external scripts available in scripts/ext that can be loaded using l.ext(...libs):

clipboard, easel, feature, jquery, jquery-ui, luxon, moment, require, sprintf, stacktrace.

The Crypto-JS libraries can be loaded using lt.cjs(...libs), and the ACE editor can be loaded using lt.ace() (no arguments for that one, as further ACE features are loaded on-demand.)
